# SharePoint JSON Formatting Samples

A collection of JSON formatting samples for SharePoint Online column, view, and form customization.

## Overview

This repository contains reusable JSON formatting examples that can be applied to SharePoint lists and libraries to enhance the visual presentation and user experience without custom code.

## Structure
```
column-samples/
├── example-1/
│   └── sample.json
├── example-2/
│   └── sample.json
...
```

Each sample is organized in its own folder for easy navigation and future documentation.

## Usage

1. Navigate to the desired sample folder
2. Copy the JSON code from the file
3. In SharePoint:
   - Go to your list/library
   - Click on a column header → Column settings → Format this column
   - Select "Advanced mode"
   - Paste the JSON code
   - Click Save
